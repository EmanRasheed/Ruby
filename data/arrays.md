# Arrays in Ruby

## What is an Array?
An array is a collection of multiple values stored in a single variable. It allows you to store and manage lists of items.

## Key Points
- Arrays can store different data types
- Elements are accessed using index (starting from 0)
- Arrays are dynamic in size

## Example
```ruby
numbers = [1, 2, 3, 4, 5]

numbers.each do |num|
  puts num
end
